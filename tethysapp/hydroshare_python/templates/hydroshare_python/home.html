Welcome to the Hydroshare app from Tethys

The features of hydroshare made available on this tethys app using the python api from hydroshare

Create a Resource

1. After installing the app on your local, enter in yur username and password in line 10 of the controllers.py file.

2. Then you can go ahead and type the path in the fpath location found in line 150 and save the changes made in the file.

3. Now click on the Create Resource button and enter the data in the boxes as mentioned and click on the 'Add+' button found int bottom right of the page.

4. You can see the new Resource created in your Hydroshare account

Add a file to the created resource

1. After successfully creating a resource in your Hydroshare account, copy the 32 element resource id that shows up on the hyperlink of the http. (for example; https://www.hydroshare.org/resource/9f0e6230688d4a95842880029d17e003/

2. Go ahead and paste that resource id in the fpath found in line 255 of controller.py file

3. Save the changes to the file and refresh the app.

4. Now click on the Add file button and enter the data in the box as mentioned and click on the 'Add+' button found in the bottom right of the page.

5. You will now the see the file added to the respective resource as desired in Hydroshare
